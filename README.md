# 🚀 AutoStream – Social-to-Lead Agent

An AI-powered conversational agent that answers product queries, detects user intent, and captures leads — built using LangGraph.

---

## 📌 Overview

This project simulates a real-world SaaS sales assistant for **AutoStream**, an AI video editing platform for content creators.

The agent:
- Responds to user queries using a knowledge base (RAG)
- Detects user intent (greeting, query, high-intent)
- Collects lead information (name, email, platform)
- Executes a tool to capture leads

---

## 🧠 Features

- ✅ Intent Detection (rule-based)
- ✅ RAG-based responses
- ✅ Lead qualification workflow
- ✅ Tool execution (mock lead capture)
- ✅ Conversational CLI interface

---

## 🛠️ Tech Stack

- Python
- LangGraph
- LangChain (core)
- Regex-based parsing
